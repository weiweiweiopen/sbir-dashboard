# SBIR 公開透明資料邊界

本 repo 以「公開行政透明」為目的，公開專案級資訊：

- 簽約版預算科目與補助／自籌金額
- 月度預估支出與累計規劃
- 已登錄實支總額與可公開的實支流水
- 專案行政狀態與查核摘要

不公開：

- 個人薪資明細、實領薪資、勞健保投保級距與個人負擔
- 身分證字號、生日、住址、電話、Email、銀行帳戶
- Simpany、Google、政府系統登入畫面、下載連結、cookie、憑證或內部截圖
- 尚未公開的合約條款、報價單、發票影本、收據影本

若未來需要引用私人系統資料，應先彙總成專案級數字並去識別化後再提交。
