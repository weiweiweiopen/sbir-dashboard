# Strategy Map

## Project thesis

The defensible center is not "solar PUF", "optical PUF", "blockchain IoT", or "light-field imaging" by itself.

The defensible center is the combination:

1. patterned DSSC photosensitive layer as material entropy source;
2. controlled illumination and I-V curve measurement;
3. feature extraction and identity reconstruction;
4. panel ID registration / verification;
5. material event signature and optional external hash anchoring.

## Product language

Use:

- DSSC-PUF material identity
- I-V signature reconstruction
- material event signature
- transparent / patterned photovoltaic authentication element

Avoid overclaiming:

- all solar PUFs
- all optical PUFs
- general blockchain IoT security
- commercial-grade REC / energy certificate equivalence
- generic light-field camera systems

## Evidence needed

- same-sample repeatability across illumination conditions;
- between-sample separability;
- I-V feature set definition: Voc, Isc, FF, Vmpp, Impp, Rs, Rsh, curve-shape descriptors;
- registration table and verification procedure;
- event log schema.
