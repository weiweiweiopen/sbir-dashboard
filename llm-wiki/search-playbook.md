# Search Playbook

Use this when web search access is available again or when a human conducts formal searching.

## Patent search queries

Run in Google Patents, Lens, Espacenet, WIPO PatentScope, USPTO, and TIPO:

- "dye sensitized solar cell" "physical unclonable function"
- DSSC PUF
- photovoltaic "physical unclonable function"
- solar cell PUF authentication
- optoelectronic PUF photovoltaic
- self powered PUF photodetector
- optical PUF material microstructure
- patterned photovoltaic authentication
- I-V curve device fingerprint
- solar panel identity verification blockchain
- material event signature photovoltaic

## CPC / IPC anchors

Start broad, then narrow:

- H04L9/32 — authentication / challenge-response;
- H04L9/08 — key management;
- G06F21 / G06F security subclasses;
- G01R — electrical measurement;
- H01G / H01L photovoltaic or photoelectric device classes;
- G02B / H04N light-field or optical imaging if optical readout is used.

## Counsel handoff

Before filing:

1. freeze candidate claim set;
2. build claim chart against closest 10–20 references;
3. ask counsel to mark novelty / inventive-step risk;
4. revise claims around the DSSC + I-V + event-signature combination.
